# mern-scheduling-app
Requirements: <br/>
React.js / MongoDB <br/>
First add a .env with the following line : <br/>
    JWT_KEY=secret <br/>
 # Then to run the App : <br/>
Start by running the server/Backend : <br/>
- ```npm install``` <br/>
- ```npm start``` <br/>
Then the frontend App : <br/>
- ```cd frontend npm install``` <br/>
- ```npm start``` <br/>
